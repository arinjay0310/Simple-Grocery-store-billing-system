# Simple-Grocery-store-billing-system
Small grocery shops often calculate bills manually which lead to errors
We need a automated billing program-
That allows the customer to select  items and quantity and calculates total bill , tax , and final amount 

Requirement Analysis And Non fucntional Requirement
Display menu of items
Take user input : Item and quantity
Calculate cost = quantity x price
Add 5% GST
Display Final bill
Easy to use

Algorithm
Define number of items and price
Show the menu
Ask user how many different items they want
For each item :
Read the name of item and quantity
Add to cart
Validate if item exists
After input ends calculate the total bill + GST
Print bill
